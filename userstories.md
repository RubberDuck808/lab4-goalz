# User Stories

---

## Epic 1 – Navigation & Exploration

- **US EP 1.1 : Achievement Metric**
  As a student, I want to measure my achievements, so that I can keep an overview of my progress.

- **US EP 1.2 : Route Navigation**
  As a player, I want to navigate a route through the arboretum, so that I can find sensors and preset locations in the correct order.

- **US EP 1.3 : Landscape Photo Capture**
  As a Trailblazer, I want to photograph predefined nature elements at preset locations, so that the box location is revealed on the map for my group.

---

## Epic 2 – Gamification & Competition

- **US EP 2.1 : Handling Pressure**
  As a player, I want to answer quiz questions under time pressure, so that I can earn more points by responding quickly and correctly.

- **US EP 2.2 : Educational Experience**
  As a student, I want to learn about nature through hint fragments and quiz questions, so that I can gain knowledge while playing the game.

- **US EP 2.3 : Points for Data Collection**
  As a Scout, I want to earn points for collecting sensor hint fragments, so that my contribution to the group is reflected in the score.

- **US EP 2.4 : Leaderboard**
  As a player, I want to see a global leaderboard with rankings, so that I can compare my performance with other players across all sessions.

- **US EP 2.5 : Team Competition**
  As a player, I want to compete against other groups during a session, so that my group is motivated to perform better than others.

- **US EP 2.6 : Team Leaderboard**
  As a player, I want to see a leaderboard of all groups within a session, so that I can track how my group ranks compared to the others in real time.

- **US EP 2.7 : Points from Multiple Activities**
  As a player, I want to earn points from both correctly answered quiz questions and speed, so that all my in-game contributions are rewarded.

---

## Epic 3 – Sensor Integration

- **US EP 3.1 : Automatic Sensor Data Collection**
  As a Scout, I want the app to automatically record a hint fragment when I interact with a sensor, so that I don't have to manually input any data.

- **US EP 3.2 : Sensor Detection**
  As a Scout, I want the app to detect nearby sensors, so that I know when I am close enough to interact with one.

- **US EP 3.3 : Sensor Data Upload**
  As a player, I want sensor interaction data to be uploaded to the server immediately, so that my group's progress is updated in real time for all members.
