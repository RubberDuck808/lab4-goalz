# ADR 001: Use ASP.NET Core (C#) for Backend

## Status
Accepted

## Context
A robust and scalable backend is required to handle sensor data and support the Office of Sustainability system.

## Decision
Use ASP.NET Core (C#) as the primary Web API framework.

## Consequences

### Positive
- High performance and strong typing
- Easy integration with Azure and PostgreSQL

### Negative
- Requires knowledge of C# and .NET ecosystem
